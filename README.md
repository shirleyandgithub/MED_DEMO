<p>这是一个医疗影像分割 + 报告生成的端到端实操的项目。<br>
基于ISIC数据集，通过UNet系模型进行影像识别; <br>
基于Baichuan-M2大模型生成报告;</p><br><br>

ISIC数据集: https://challenge.isic-archive.com/data/#milk10k<br><br>



**<p>业务逻辑<br>**
1.数据准备:读取Metadata.csv(基础信息)和Supplement.csv(病理结果); 通过isic_id将两张表合并(Inner Join)，确保得到的病例既有年龄、性别等基本信息，同时也有确诊 的结果;<br><br>
2. 视觉分析:调用UNet模型; 将图像转换成Tensor格式，送入UNet模型，模型输出一个Mask(这个Mask本质是一张黑白图，白色 表示病灶、黑色表示健康皮肤)，将Mask中白色像素数除以总像素数，得到病灶占比数值，将这个结 果转化成一段自然语言描述，写死成一句话:“病灶区域分割完成，病变占比约24.5%，边缘检测显 示不规则。”;<br><br>
3. 多模态提示词融合:将“视觉”和“数据”两路信息拼接到同一个提示词中; 即上一步UNet生成的百分比和写死那句话，以及CSV文件中读取的基本信息和病理结果;<br><br>
4. 认知推理与生成:发送HTTP请求调取Baichuan-M2的API;<br><br>
5. 结果渲染:终端输出 + 生成报告(图像); 利用Pillow库将医疗大模型生成的文本保存成png图像，存在当前文件夹下;
</p>
